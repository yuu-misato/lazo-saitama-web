# Lazo 埼玉版 公式サイト

埼玉のペット家族向けフリーペーパー「Lazo 埼玉版」の公式Webサイト（制作中）

## 運営
株式会社スペーススタッフ / 代表取締役 田中 留美

## 技術スタック
- 静的HTML / CSS / JavaScript（フレームワーク不使用）
- Google Fonts: Noto Sans JP / Zen Kaku Gothic New / Inter

## ページ構成
- `index.html` — トップページ
- `about.html` — Lazoとは / 運営会社
- `latest.html` — 最新号・バックナンバー
- `kotogoto.html` — お困りごと解決
- `area.html` — うちの町のLazo
- `community.html` — うちの子自慢
- `disaster.html` — ペット防災
- `shops.html` — 設置店マップ
- `shop-register.html` — 店舗無料掲載申込
- `ad.html` — 広告掲載
- `contact.html` — お問い合わせ

## ローカルプレビュー
```bash
python3 -m http.server 8000
# → http://localhost:8000/
```
